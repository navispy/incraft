A lightweight plugin with simple one page setting. This plugin enables you to add custom html on either header, or footer or both for all pages.

== Description ==

Header Footer Custom Html plugin by Enweby is a light weight plugin provides easy interface to add custom html code in header or footer or even in both header and footer, choice is yours.
The setting is global so the code will be available on all the pages.

This plugin when used with correct css can be a great tool to easily add floating banner, promotional buttons, floating notification, and many more. Possibility is endless.

**Header Footer Custom Html can be used as following tool or functionality**

* Beautiful Promotional Banners in the header or footer
* Custom css code allowed right in the admin settings.
* Floating section in either header or footer.
* Site wise notification section like covid 19 alert
* Floating banner on the right or left side

== Installation ==

**This section describes how to install the plugin and get it working.**

= AUTOMATIC INSTALLATION (EASIEST WAY) =

To do an automatic install of 'Header Footer Custom Html', log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
In the search field type 'Header Footer Custom Html' by Enweby.
Once you have found it you can install it by simply clicking "Install Now" and then "Activate".

= MANUAL INSTALLATION =

**Installing through WordPress Dashboard**

* Download header-footer-custom-html.zip
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select header-footer-custom-html.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

**Using FTP**

* Download header-footer-custom-html.zip
* Extract the header-footer-custom-html.zip to your computer
* Upload the header-footer-custom-html directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard
